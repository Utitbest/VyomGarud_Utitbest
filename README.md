VyomGarud - Web Developer Landing Page
Overview

This is a modern, responsive landing page for VyomGarud, a UAV/drone systems company. The design is dark, sleek, and slightly military/futuristic to convey precision, reliability, and advanced autonomy.

The page showcases:

Hero section with full-screen background video

Company mission / About section

Capabilities / Product cards

Highlights features

Contact / Footer section

All elements feature smooth scroll-triggered animations for a dynamic, engaging experience.

Technologies Used

React – Frontend framework

Tailwind CSS – Styling and layout

Framer Motion – Scroll animations & hero effects

Video assets – Background hero video

Features

Responsive Design – Works on mobile, tablet, and desktop

Hero Video – Dimmed full-screen video with subtle zoom effect

Scroll Animations – Fade-in and slide-up for cards and sections

Interactive Buttons – Hover scale effect for CTAs

Reusable Components – Easy to maintain and extend

Setup Instructions

Clone the repository:

git clone <repo-url>
cd VyomGarud_Utitbest


Install dependencies:

npm install


Run the development server:

npm run dev


Open http://localhost:3000
 to view in browser.

Design Notes

Colors: Dark background (#0c0c0c / #0f0f0f), Orange accent (#ff7b00), White/Gray text

Fonts: Poppins (headings), Inter (body)

Animations: Framer Motion for scroll fade-ins and hero video zoom

Layout: Tailwind CSS grid and flex utilities for responsive sections
